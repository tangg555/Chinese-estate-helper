# Chinese-estate-helper

[![License: GPL](https://img.shields.io/badge/License-GPL-yellow.svg)](https://opensource.org/licenses/GPL)

## 介绍
爬取房产网站数据用作自用分析。

## 安装教程

代码开发版本为Python 3.6，请自行安装。

## 使用说明

下载代码，打开终端切换至代码的根目录。
运行
```python3
pip install requirements.txt
cd tasks
python basic_task.py
```
